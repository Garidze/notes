1. установка файлового менеджера:

		sudo apt -y install mc (sudo apt-get -y install mc);


2. создать пароль, если не установлен для root:

		passwd root;


3. Изменить параметр в /etc/ssh/sshd_config :

		PermitRootLogin yes;
		
		
4. Перезапускаем ssh server:

		systemctl restart ssh || systemctl restart sshd
